# ZuoChuan

此網頁為臺中教育大學 通識課程 經典的智慧 期末作業

網頁功能為利用左傳的段落，查詢該段落相關聯的國家

亦或是選取一個或多個國家，顯示選取國家有參與的事件

GitHub Pages網址:

[https://adt109119.github.io/ZuoChuan/](https://adt109119.github.io/ZuoChuan/)
